# Awesome Feed

[![Build Status](https://travis-ci.com/peehaa/feedr.svg?branch=master)](https://travis-ci.com/peehaa/feedr)

Converts any resource into an RSS feed

## Requirements

- PHP7.2
- NodeJS
- PostgreSQL

## Installation

    composer install
    
*This will guide your through the installation script.*

## Development

    npm run dev
