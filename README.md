Converts any resource into an RSS feed

## Requirements

- PHP7.2
- NodeJS
- PostgreSQL
- Redis

## Installation

    composer install
    
*This will guide your through the installation script.*

## Start the websocket server and queue

    php ./bin/server.php

## Development

    npm run dev
